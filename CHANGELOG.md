# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.2.0] - 2026-03-24

### Added
- Configurable label text for the default payment method (shown after method title in checkout)

### Changed
- Moved admin configuration from custom section to payment methods configuration
- Requires PHP 8.4+

## [2.1.0] - 2025-01-01

### Changed
- Code modernization with strict types and PHP 8.1+ features

## [2.0.0] - 2024-01-01

### Changed
- Requires PHP 8.1+
- Block rewrite uses `final` class

### Removed
- Dropped support for PHP 7.x

## [1.0.0] - 2016-01-01

### Added
- Initial release
- Admin configuration to select default payment method
- Automatic pre-selection in onepage checkout
